Bootstrap project for [rtpstalk](https://github.com/pinorobotics/rtpstalk)

# Build

Using Gradle:

``` bash
gradle clean build
```

# Usage

Run Subscriber:

``` bash
java -cp rtpstalkexample.jar id.bootstrap.rtpstalk.SubscriberApp
```

Run Publisher:

``` bash
java -cp rtpstalkexample.jar id.bootstrap.rtpstalk.PublisherApp
```
